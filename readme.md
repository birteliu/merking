# 文件說明

專案說明

========================

# 應用第三方

Boostrap 5.3.3
Fontawesome 6.7.2

========================

# 使用方法

1. 安裝 npm install
2. 一般切版 使用 gulp dev
3. 專案內執行 使用 gulp watch

========================

# 應用場景

0. \_demo.html 範本檔
1. index.html 首頁
2.

========================

# 備註

- 若不要複製到 wwwroot ，可將 tasks/copy.js 的 .pipe(gulp.dest("../../wwwroot")) 這行註解掉
- icon 透過 cdn 方式引入，直接替換 style.scss 裡的 //base 連結即可
